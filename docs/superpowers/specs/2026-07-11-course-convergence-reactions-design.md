# Course Convergence Reactions Design

## Goal

마지막 랜딩 모션에서 장소 카드 3개가 화면 중앙의 하루 코스 폴더로 모이고, 완성 직후 친구와 큐레이터의 반응이 폴더 주변 여러 방향에서 나타난다.

## Sequence

1. 식당, 카페, 도예실 카드가 화면의 서로 다른 가장자리에서 등장한다.
2. 세 갈래 초록색 연결선이 중앙을 향해 그려진다.
3. 카드 3개가 중앙으로 이동하며 축소되고 하루 코스 폴더로 합쳐진다.
4. 폴더 주변에 `여기 저장했어요`, `다음 주에 같이 가자!`, `이 코스 진짜 좋아요`, `♥ 128`, `코스 저장 완료`가 프로필과 함께 흩어져 나타난다.
5. 모바일에서도 장면 내부 가로 스크롤이나 잘림이 없어야 한다.

## Constraints

- 기존 랜딩 문구와 알림신청 버튼은 수정하지 않는다.
- 길찾기 UI는 추가하지 않는다.
- 기존 최적화 AVIF 사진과 프로필을 재사용한다.
