# Instagram 캐러셀 단순화 디자인

## 목적

Doripe Instagram 콘텐츠를 사진 중심으로 단순화하고, 모든 게시물의 마지막 장에서 Doripe 앱으로 자연스럽게 연결한다.

## 확정 규칙

### 한 장소·행사 콘텐츠

- 첫 장에는 기존처럼 장소명 또는 행사명과 핵심 문구를 넣는다.
- 첫 장과 마지막 공통 장 사이의 사진 장에는 본문, 위치, 날짜, 출처 문구를 넣지 않는다.
- 사진 장에는 실제 웹 사진과 작은 Doripe 심볼만 둔다.
- 사진 출처와 방문 정보는 이미지 위가 아니라 package의 `caption.txt`, `sources.txt`, `review.txt`에 기록한다.

### CTA 제거

- 모든 템플릿에서 `보내주세요`, `저장하세요` 같은 직접 행동 문구를 제거한다.
- 템플릿 계약과 자동 검수에서도 `slot:cta`를 필수 항목으로 요구하지 않는다.
- 캡션에도 직접적인 send/save CTA를 넣지 않는다.

### 공통 마지막 장

- 모든 콘텐츠 유형의 마지막에 1080×1350 공통 브랜드 장을 한 장 추가한다.
- 배경은 검은색 계열로 고정한다.
- 중앙에는 Doripe 앱 휴대폰 목업을 배치한다.
- 질문형 문구는 상단 안전영역 안에 두고 중앙 정렬한다.
- 하단에는 녹색 Doripe 심볼과 `Doripe.` 워드마크를 함께 둔다.

## 마지막 장 문구 규칙

문구는 매번 콘텐츠 유형과 주제에 맞춰 바꾸되, 질문 하나만 사용한다.

- 전시·행사: `이 전시 다음에 갈 장소가 궁금하다면?`
- 단일 장소: `이곳과 함께 갈 장소가 궁금하다면?`
- 장소 모음: `오늘의 취향을 더 발견하고 싶다면?`
- 코스: `내 취향으로 새로운 하루를 만들고 싶다면?`

문구가 길면 의미를 유지한 채 줄이고, 단어 중간 줄바꿈은 허용하지 않는다. 마지막 줄의 브랜드 표기는 항상 `Doripe.`로 고정한다.

## 자동화 동작

- 콘텐츠 조사와 사진 수집 방식은 기존대로 국내 실제 웹 사진만 사용한다.
- 생성기는 콘텐츠 유형에 맞는 질문 문구를 선택한다.
- 한 장소·행사 콘텐츠의 중간 장은 사진과 심볼만 교체한다.
- 컬렉션과 코스는 필요한 정보형 본문을 유지할 수 있지만 직접 CTA는 제거한다.
- 모든 유형에 공통 마지막 장을 붙인 뒤 1080×1350, 안전영역, 줄바꿈, 사진 출처 기록을 검수한다.

## 검수 기준

- 한 장소·행사 중간 장에 Doripe 심볼 외 텍스트가 없어야 한다.
- 어느 장에도 `slot:cta` 또는 직접 send/save 문구가 없어야 한다.
- 모든 게시물의 마지막 장이 검은 배경, 휴대폰 목업, 질문형 문구, Doripe 브랜드 표기를 포함해야 한다.
- 마지막 장 질문은 해당 콘텐츠와 의미가 맞아야 한다.
- 사진 권리와 개인정보 경고는 기존과 같이 package 파일에 남아야 한다.
- Instagram 로그인과 자동 게시는 계속 금지한다.

## 적용 범위

- Figma의 `place_event`, `collection`, `route` 승인 템플릿
- 템플릿 계약과 자동 검수
- 일일 실행 가이드
- 2026-07-14 드라이런 결과물 두 개

Brain 문서는 이번 변경에서 수정하지 않는다.
